# Search Strategies Guide

## General Search Techniques

### 1. Keyword Development
- **Root Keywords:** Core terms directly related to topic
- **Synonyms:** Alternative terms (e.g., "AI" vs "artificial intelligence") 
- **Related Terms:** Adjacent concepts (e.g., "machine learning" for AI research)
- **Industry Jargon:** Specialized terminology used by experts
- **Boolean Operators:** AND, OR, NOT for complex queries
- **Phrase Searching:** "exact phrases" in quotes
- **Wildcard Searching:** Use * for partial matches

### 2. Source Diversification
- **Academic Sources:** Google Scholar, JSTOR, PubMed, ArXiv
- **News Sources:** Google News, Reuters, AP, BBC, industry publications
- **Government Sources:** .gov sites, official statistics, regulatory documents
- **Industry Sources:** Trade publications, company reports, analyst reports
- **Social Sources:** Reddit, Twitter, LinkedIn, industry forums
- **International Sources:** Non-English sources for global perspective

### 3. Search Engine Optimization
- **Google Advanced Operators:**
  - `site:example.com` - Search within specific domain
  - `filetype:pdf` - Find specific file types
  - `inurl:research` - Keywords in URL
  - `intitle:"market research"` - Keywords in title
  - `daterange:` - Specific time periods
  - `related:example.com` - Similar websites

- **Bing Advanced Operators:**
  - `contains:pdf` - Files containing specific content
  - `language:en` - Language-specific results
  - `location:US` - Geographic targeting
  - `feed:` - RSS feed searches

## Specialized Research Strategies

### Market Research
1. **Company Analysis:**
   - Annual reports (10-K, 10-Q filings)
   - Press releases and earnings calls
   - LinkedIn company pages and employee counts
   - Glassdoor for company culture insights

2. **Industry Analysis:**
   - IBISWorld, Euromonitor, Mintel reports
   - Trade association publications
   - Market research firm reports (Gartner, Forrester)
   - Patent databases for innovation trends

3. **Competitive Intelligence:**
   - Competitor websites and product pages
   - Social media presence and engagement
   - Job postings for strategic hiring insights
   - Customer review sites and feedback

### Technical Research
1. **Documentation Search:**
   - Official documentation sites
   - GitHub repositories and README files
   - Stack Overflow and developer forums
   - API documentation and specifications

2. **Best Practices:**
   - Industry blogs and thought leadership
   - Conference presentations and videos
   - Case studies and implementation examples
   - Open source project analysis

3. **Trend Analysis:**
   - Google Trends for search interest
   - GitHub trending repositories
   - Survey reports (Stack Overflow, JetBrains)
   - Technology adoption reports

### Academic Research
1. **Literature Search:**
   - Systematic keyword variations
   - Citation tracking (forward and backward)
   - Author following for expertise mapping
   - Conference proceeding searches

2. **Quality Assessment:**
   - Journal impact factors
   - Citation counts and h-index
   - Peer review status
   - Replication studies

## Quality Control Strategies

### Source Validation
1. **Authority Indicators:**
   - Author credentials and expertise
   - Publication reputation and peer review
   - Institutional affiliation
   - Citation by other reputable sources

2. **Currency Assessment:**
   - Publication or update dates
   - Relevance to current context
   - Superseding information availability
   - Version control for evolving topics

3. **Bias Detection:**
   - Funding source identification
   - Potential conflicts of interest
   - Language tone and objectivity
   - Statistical methodology review

### Cross-Validation Techniques
1. **Triangulation:**
   - Multiple independent sources
   - Different methodological approaches
   - Various perspective representation
   - Confirmation across source types

2. **Fact-Checking:**
   - Primary source verification
   - Statistical accuracy validation
   - Quote and attribution verification
   - Timeline and sequence checking

## Error Recovery Strategies

### Access Issues
1. **Site Blocking:**
   - Try different search approaches
   - Use cached versions (Wayback Machine)
   - Find alternative sources
   - Use proxy or VPN if appropriate

2. **Rate Limiting:**
   - Implement delays between requests
   - Use rotating search strategies
   - Batch requests efficiently
   - Respect robots.txt guidelines

### Information Gaps
1. **Insufficient Data:**
   - Broaden search criteria
   - Try related keywords
   - Look for proxy indicators
   - Consider primary research needs

2. **Conflicting Information:**
   - Document all conflicting sources
   - Analyze methodology differences
   - Seek authoritative arbitration
   - Note limitations in conclusions
